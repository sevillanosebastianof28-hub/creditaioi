### 2.1 Ineligible Dispute Scenarios

- DOFD is accurate

## 4. REQUIRED EVIDENCE

### 4.1 For DOFD Disputes
- Credit reports from all 3 bureaus showing different DOFDs
- Original creditor statements with payment history
- Documentation showing when account actually became delinquent

### 4.2 For Balance Disputes
- Settlement agreement showing $0 balance
- Payment confirmations
- Creditor statements post-settlement
- Bank records of settlement payment

### 4.3 For Status Disputes
- Settlement letter
- Paid-in-full letter
- Bankruptcy discharge papers
- Account statements showing current status

### 4.4 For "Not My Account" Disputes
- Identity theft report (FTC.gov)
- Police report
- Fraud affidavit
- Documentation of fraud

## 5. EXPECTED BUREAU RESPONSE TYPES

### 5.1 Possible Outcomes
| Response | Meaning | Next Steps |
|----------|---------|------------|
| Deleted | Charge-off removed | Success - monitor |
| Updated | Information corrected | Verify all corrections made |
| Verified | Creditor confirmed accuracy | Evaluate escalation |
| Modified | Partial changes | Review what changed |

### 5.2 Understanding "Verified" Response
- Creditor confirmed information to bureau
- Does NOT mean thorough investigation occurred
- May warrant escalation with evidence
- Consider direct creditor dispute

---

## 6. ESCALATION RULES

### 6.1 When to Escalate
Escalation appropriate when:
- Have evidence of specific inaccuracy
- Bureau didn't consider provided evidence
- New information obtained
- Creditor failed to update after settlement

### 6.2 Escalation Path
1. **Round 1**: Standard bureau dispute
2. **Round 2**: Bureau dispute with documentation + direct creditor dispute
3. **Round 3**: Bureau escalation (executive offices)
4. **Round 4**: CFPB complaint if evidence supports

### 6.3 Escalation NOT Appropriate
- Charge-off is accurate
- No evidence of inaccuracy
- Disputing simply because it's negative
- Previous CFPB on same issue

## 8. STRATEGY RECOMMENDATIONS

### 8.1 For Recent Charge-Offs (< 2 years)
1. Document DOFD immediately
2. Compare across all 3 bureaus
3. Identify specific inaccuracies
4. Consider negotiating update if paying
5. Request "paid charge-off" status if settling

### 8.2 For Older Charge-Offs (2-5 years)
1. Verify DOFD hasn't been changed
2. Check for reporting limit violations
3. Confirm balance is accurate
4. Look for duplicate reporting with collections

### 8.3 For Near-Limit Charge-Offs (6-7 years)
1. Calculate exact 7-year date from DOFD
2. Document if past limit
3. Dispute if beyond 7 years
4. Monitor for automatic aging off

### 8.4 For Paid/Settled Charge-Offs
1. Confirm status reflects payment
2. Verify balance shows $0 or settlement amount
3. Dispute if still showing balance owed
4. Keep settlement documentation indefinitely

## 9. CHARGE-OFF VS COLLECTION

### 9.1 Key Differences
| Factor | Charge-Off | Collection |
|--------|-----------|------------|
| Who Reports | Original Creditor | Collection Agency |
| Account Type | Original account | Separate tradeline |
| DOFD | From original account | Same as original |
| Reporting Period | 7 years from DOFD | 7 years from original DOFD |

### 9.2 Duplicate Reporting Issues
Both should NOT report:
- Full balance on both
- Same account as two tradelines with balances

Both CAN report:
- Charge-off with $0 balance + collection with balance
- Or charge-off transferred to collections

### 9.3 What to Dispute
- If both show balance, one should be $0
- DOFD should match on both
- Only one should show as "owed"
